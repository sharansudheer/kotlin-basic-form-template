## Template App

### This is for the freelancers/university buddies out there

This is a Java based Native app template following The MVVM Architecture, one Main Activity and sub fragraments. One main activity and many sub fragments works better with Kotlin-coroutines and kotlin navgraph.

2. Data Store integration
3. Firebase Fragment
4. Recycler View-ViewModel for dynamic data updation
5. Material UI 3
